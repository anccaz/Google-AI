# Google-AI

Artifical Intelligence: branch of computer science 
- The theory that methods to build machines that think like humans

Machine Learning: subfield of artificial intelligence
- Program or system that trains a model from input data

* Machine Learning Models 
Supervised: Labeled data
- Labeled datasets comes with a tag 
- Used to predict for new data
- Higher accuracy 
- Use cases: spam filter, weather forecast

Unsupervised: Unlabeled data
- No tag
- Used to explore data to find patterns
- Use cases: Customer segmentation, anomaly detection
